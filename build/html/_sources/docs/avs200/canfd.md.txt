#CANFD 

![canfd](../../_static/canfd.png)

## 딥스위치(DIP Switch)

- **1번 스위치 ON/OFF**: can0 종단 저항(Termination) ON/OFF
- **2번 스위치 ON/OFF**: can1 종단 저항(Termination) ON/OFF

## 입력 하이 임계값

- **5V 기준**: 3.5V 이상
- **3.3V 기준**: 2.0V 이상

## 입력 로우 임계값

- **5V 기준**: 1.5V 이하
- **3.3V 기준**: 0.8V 이하

S/W 설정은 [인터페이스 안내서의 CAN 섹션](interface.md#can-interface-section)을 참조하세요.
